## Quick Start

1. Clone the repository:
```bash
git clone https://github.com/penomovu/penomovu-bio.git
cd penomovu-bio
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
